# GTM Revenue Tracking Tool

Herramienta interna del equipo de Marketing / Go-To-Market de [Dcycle](https://dcycle.io) para el seguimiento y la atribución de revenue generado por las acciones de GTM.

> ⚠️ **Estado: en desarrollo inicial.** El repositorio acaba de crearse y todavía no contiene código. Este README sirve como punto de partida y se irá actualizando a medida que el proyecto avance.

## Objetivo

Centralizar el tracking de ingresos asociados a la actividad de marketing y ventas, para poder responder a preguntas como:

- ¿Qué canales y campañas generan pipeline y revenue real?
- ¿Cómo evoluciona el revenue atribuido a cada motion de GTM (inbound, outbound, partners…)?
- ¿Qué retorno tienen las acciones de marketing a lo largo del funnel?

## Alcance previsto

- Ingesta de datos de las fuentes de GTM (CRM, herramientas de marketing, facturación).
- Modelo de atribución de revenue por canal, campaña y motion.
- Dashboards / reporting para el equipo de Marketing y Revenue.

*(Pendiente de concretar: stack tecnológico, fuentes de datos definitivas y modelo de atribución.)*
